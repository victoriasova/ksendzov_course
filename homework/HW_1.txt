1) Посмотреть где я - pwd
2) Создать папку - mkdir folder
3) Зайти в папку - cd foldername
4) Создать 3 папки - mkdir FN1 FN2 FN3
5) Зайти в любоую папку cd foldername/foldername2 (путь)
6) Создать 5 файлов (3 txt, 2 json) touch File1.txt File2.txt File3.txt File4.json File5.json
7) Создать 3 папки - mkdir FN1 FN2 FN3
8. Вывести список содержимого папки - ls -ls или ls или ls -l
9) + Открыть любой txt файл - cat filename.txt или открываем vim filename.txt
10) + написать туда что-нибудь, любой текст. - cat >> filename.txt вводим тест или открываем vim filename.txt жмем i, пишем
11) + сохранить и выйти. -enter i ctrl+c для vim - esc потом :wq
12) Выйти из папки на уровень выше - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку.  - mv FN1/File1 File2 FN2/File1 File2
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.  - cp FN1/File1 File2 FN2/File1 File2
15) Найти файл по имени - find -name "filename1.txt"
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
17) вывести несколько первых строк из текстового файла - head -n 5/10... filename.txt
18) вывести несколько последних строк из текстового файла tail -n 5/10... filename.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
20) вывести дату и время - date
